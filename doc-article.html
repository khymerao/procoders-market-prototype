Connect CF7 to HubSpot

Send Contact Form 7 submissions to HubSpot contacts and deals without Zapier. This guide covers installation, API authentication, field mapping, and common production issues.

Plugin: Connect CF7 to HubSpot Last updated: June 2026 WP 5.3+ · PHP 8.0+

Prerequisites

Before installing, confirm your stack meets the requirements and you have HubSpot credentials ready.

  • WordPress 5.3 or newer with Contact Form 7 active
  • PHP 8.0+ with curl and json extensions enabled
  • A HubSpot account with Marketing Hub or Sales Hub (private app access)
  • Pro license key from ProCoders Market if you purchased the Pro tier
Free vs Pro The WordPress.org build syncs email and name fields only. Pro unlocks custom property mapping, deal creation, and retry queues.

Install the plugin

Download the latest release from your ProCoders account or install the free build from WordPress.org, then activate.

  1. In WordPress admin, go to Plugins → Add New → Upload Plugin and select the .zip archive.
  2. Activate Connect CF7 to HubSpot and open Settings → CF7 HubSpot.
  3. Paste your ProCoders license key on the License tab if you purchased Pro.

Alternatively, deploy via WP-CLI on staging or production:

Terminal wp plugin install connect-cf7-hubspot --activate wp option update cf7hs_license_key "YOUR-LICENSE-KEY"

HubSpot connection

Create a private app in HubSpot and paste the access token into the plugin settings screen.

  1. In HubSpot, open Settings → Integrations → Private Apps and create an app named WordPress CF7.
  2. Grant scopes: crm.objects.contacts.write, crm.objects.deals.write (Pro), and crm.schemas.contacts.read.
  3. Copy the access token and paste it into CF7 HubSpot → Connection → Access token.
  4. Click Test connection. A green status badge confirms the token is valid.
Token rotation If you regenerate the HubSpot token, update it in WordPress immediately. Queued submissions fail until the new token is saved.

Field mapping

Map each CF7 field to a HubSpot contact property. Pro supports deal pipelines and custom objects.

On the Field mapping tab, select your CF7 form from the dropdown. For each form tag, choose the HubSpot property:

  • your-email → email (required)
  • your-name → firstname + lastname split rule
  • Custom tags → HubSpot custom properties (Pro)

Advanced mappings can be declared in wp-config.php for multisite or CI-managed configs:

wp-config.php or theme functions add_filter( 'cf7hs_field_map', function ( $map, $form_id ) { if ( $form_id === 42 ) { $map['your-company'] = 'company'; $map['your-phone'] = 'phone'; } return $map; }, 10, 2 );

Deal creation (Pro): enable Create deal on submit, pick a pipeline and stage, and map a deal name from a CF7 field or static string.

Test & troubleshoot

Submit a test entry from the front end with a unique email address. Check HubSpot contacts within 30 seconds.

  • Submission stuck in queue — open CF7 HubSpot → Logs and retry failed rows after fixing the token or mapping.
  • Duplicate contacts — enable Update existing by email on the Connection tab.
  • 403 from HubSpot — verify private app scopes include write access for contacts.
  • CF7 mail still sends but HubSpot empty — confirm the form ID is selected on the mapping tab; unmapped forms are ignored by design.

Enable debug logging temporarily:

wp-config.php define( 'CF7HS_DEBUG', true );

Logs write to wp-content/uploads/cf7hs-debug.log. Disable the constant after diagnosis.
